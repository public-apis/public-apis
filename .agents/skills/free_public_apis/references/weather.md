# Weather APIs

This file contains free, No-Auth, HTTPS APIs for this category.

| API Name | Description | Documentation URL |
|---|---|---|
| AviationWeather | NOAA aviation weather forecasts and observations | https://www.aviationweather.gov/dataserver |
| Hong Kong Obervatory | Provide weather information, earthquake information, and climate data | https://www.hko.gov.hk/en/abouthko/opendata_intro.htm |
| MetaWeather | Weather | https://www.metaweather.com/api/ |
| Open-Meteo | Global weather forecast API for non-commercial use | https://open-meteo.com/ |
| openSenseMap | Data from Personal Weather Stations called senseBoxes | https://api.opensensemap.org/ |
| RainViewer | Radar data collected from different websites across the Internet | https://www.rainviewer.com/api.html |
| US Weather | US National Weather Service | https://www.weather.gov/documentation/services-web-api |
| weather-api | A RESTful free API to check the weather | https://github.com/robertoduessmann/weather-api |
