# Sports & Fitness APIs

This file contains free, No-Auth, HTTPS APIs for this category.

| API Name | Description | Documentation URL |
|---|---|---|
| balldontlie | Balldontlie provides access to stats data from the NBA | https://www.balldontlie.io |
| City Bikes | City Bikes around the world | https://api.citybik.es/v2/ |
| Ergast F1 | F1 data from the beginning of the world championships in 1950 | http://ergast.com/mrd/ |
| Football (Soccer) Videos | Embed codes for goals and highlights from Premier League, Bundesliga, Serie A and many more | https://www.scorebat.com/video-api/ |
| Football Standings | Display football standings e.g epl, la liga, serie a etc. The data is based on espn site | https://github.com/azharimm/football-standings-api |
| NBA Stats | Current and historical NBA Statistics | https://any-api.com/nba_com/nba_com/docs/API_Description |
| NHL Records and Stats | NHL historical data and statistics | https://gitlab.com/dword4/nhlapi |
| Oddsmagnet | Odds history from multiple UK bookmakers | https://data.oddsmagnet.com |
| OpenLigaDB | Crowd sourced sports league results | https://www.openligadb.de |
| Sport List & Data | List of and resources related to sports | https://developers.decathlon.com/products/sports |
| Sport Places | Crowd-source sports places around the world | https://developers.decathlon.com/products/sport-places |
| Squiggle | Fixtures, results and predictions for Australian Football League matches | https://api.squiggle.com.au |
