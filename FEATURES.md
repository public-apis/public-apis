# Project Features & Tech Stack

## Features

### 1. Search & Filtering

- Інтуїтивний пошук API по назві або ключових словах.
- Фільтри за категоріями, способом авторизації, HTTPS та CORS.
- Можливість комбінувати декілька фільтрів.

### 2. API Status Checker

- Автоматична перевірка доступності API.
- Відображення статусу у реальному часі (Online/Offline/Unknown).

### 3. Pull Request via Form

- Форма для додавання нового API до репозиторію `public-apis`.
- Автоматичне створення Pull Request на GitHub через GitHub API.
- Валідація введених даних перед відправкою.

### 4. User Interface

- Зручний та адаптивний інтерфейс для пошуку і фільтрації.
- Динамічні списки та таблиці з API.

### 5. Backend Services

- REST API для перевірки статусу зовнішніх API.
- Обробка форм для створення Pull Request.
- Кешування результатів для зменшення навантаження на API.

---

## Tech Stack

### Frontend

- **React** – основна бібліотека для UI.
- **TailwindCSS / shadcn/ui** – стилізація та компоненти.

### Backend

- **NestJS** – основний серверний фреймворк.
- **Node.js** – runtime.
- **Axios** – для запитів до зовнішніх API.
- **GitHub API** – для створення Pull Request.

## Summary

Цей проект надає користувачам можливість швидко знаходити потрібні публічні API, перевіряти їх доступність і додавати нові API через форму, яка автоматично створює Pull Request в репозиторії `public-apis`. Все це реалізовано на сучасному стеку фронтенд + бекенд з акцентом на швидкість та зручність користування.
