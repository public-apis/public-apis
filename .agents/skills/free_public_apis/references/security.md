# Security APIs

This file contains free, No-Auth, HTTPS APIs for this category.

| API Name | Description | Documentation URL |
|---|---|---|
| Classify | Encrypting & decrypting text messages | https://classify-web.herokuapp.com/#/api |
| Dehash.lt | Hash decryption MD5, SHA1, SHA3, SHA256, SHA384, SHA512 | https://github.com/Dehash-lt/api |
| EmailRep | Email address threat and risk prediction | https://docs.emailrep.io/ |
| Escape | An API for escaping different kind of queries | https://github.com/polarspetroll/EscapeAPI |
| FilterLists | Lists of filters for adblockers and firewalls | https://filterlists.com |

| National Vulnerability Database | U.S. National Vulnerability Database | https://nvd.nist.gov/vuln/Data-Feeds/JSON-feed-changelog |
