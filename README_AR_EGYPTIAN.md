# موسوعة الـ APIs العامة - النسخة العربية المصرية 🇪🇬

## مقدمة: إيه القصة أصلاً؟

المشروع ده عبارة عن "دليل ضخم" أو "كنز" جامع لآلاف الـ **APIs** المجانية والمتاحة لأي حد يستخدمها. 
يعني إيه API؟ تخيلها زي "الجرسون" في المطعم.. أنت بتطلب منه أكلة (بيانات)، وهو بيروح للمطبخ (السيرفر) ويجيبالك الطلب. 
المشروع ده بيعمل فهرسة لكل "الجراسنة" دول عشان تعرف مين فيهم بيعمل إيه، وإزاي تقدر تتواصل معاه.

المشروع ده بيشغله مجتمع من المطورين المتطوعين وشركات زي **APILayer**، وهدفه يساعد المبرمجين وصناع التطبيقات إنهم يلاقوا أدوات جاهزة يركبوها في مشاريعهم من غير وجع دماغ.

## ملاحظات مهمة جداً قبل الاستخدام:

1.  **مفتاح API (apiKey):** بعض الخدمات دي بتطلب منك "كود سري" عشان تشتغل. الغالبية بتديهولك مجاناً بس لازم تسجل في موقعهم الأول.
2.  **OAuth:** دي طريقة دخول معقدة شوية، بتخليك تدخل بحسابك في جوجل أو فيسبوك عشان تاخد الصلاحية.
3.  **CORS:** دي حاجة تقنية بتسمح للموقع بتاعك إنه يكلم الـ API ده من المتصفح مباشرة. لو مكتوب "Yes" يبقى كله تمام، لو "No" ممكن تحتاج وسيط.
4.  **HTTPS:** يعني الاتصال آمن ومشفّر (قفل أخضر)، ودي حاجة ضرورية جداً عشان بياناتك متتسرقش.
